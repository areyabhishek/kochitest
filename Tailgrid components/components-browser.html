How to Use Components

  1. Click on any category above to browse its components
  2. Open any .html file to view the component code
  3. Copy the HTML code from the component file
  4. Paste it into your landing page (page1.html, page2.html, etc.)
  5. Customize colors, text, and styling as needed
  6. Save and refresh your browser to see the changes

Pro Tip:

Mix and match components from different categories to create unique landing pages. Each component is self-contained and works independently!
