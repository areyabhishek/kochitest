# Teamtrip - Unforgettable Offsites for Indian Companies

A modern website for Teamtrip, an Indian company specializing in planning unique offsite events and corporate experiences.

## Features

- **Hero Section** - Eye-catching introduction with call-to-action
- **Services** - Showcase of offsite planning, corporate events, conferences, and workshops
- **Destinations** - Interactive carousel of Indian and international destinations
- **Features** - Highlighting unique advantages and benefits
- **Pricing** - Transparent pricing information
- **Testimonials** - Customer reviews and feedback
- **FAQ** - Frequently asked questions with expandable answers
- **Contact Form** - Easy-to-use contact form for proposals

## Tech Stack

- **Next.js 14** - React framework for production
- **TypeScript** - Type-safe development
- **Tailwind CSS** - Utility-first CSS framework
- **Responsive Design** - Mobile-first approach

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build for Production

```bash
npm run build
npm start
```

## Customization

The website is fully customizable. You can:
- Update colors in `tailwind.config.js`
- Modify content in each component file
- Add new sections or features
- Customize the Indian destinations list
